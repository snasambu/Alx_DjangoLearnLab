# LibraryProject

## Overview
LibraryProject is the initial setup for a Django project. It serves as the foundation for developing Django applications, demonstrating the default project structure and configuration.

## Features
- Django 5.2.5 installed and configured
- Default project structure created with `django-admin startproject`
- Includes:
  - `manage.py` for project commands
  - `settings.py` for project configuration
  - `urls.py` for URL routing
  - `wsgi.py` and `asgi.py` for server interfaces
- Development server verified at http://127.0.0.1:8000/
